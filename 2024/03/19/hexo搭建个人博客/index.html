hexo搭建个人博客

发表于2024-03-19|更新于2024-03-19|嵌入式
|字数总计:483|阅读时长:1分钟|阅读量:

1.下载nodejs

网址：nodejs.org

2.安装nodejs

打开下载的安装包，一直next就行

3.打开终端

1.查看版本

2.安装淘宝的cnpm管理器

3.查看cnpm版本

4.安装hexo框架

5.查看hexo版本

6.查看当前路径

7.创建blog目录

linux下：mkdir blog直接创建

window下：

  • 如果一开始在C盘下，输入D: 进入D盘
  • 在D盘下创建：mkdir blog

8.生成博客，初始化博客

  • linux下：sudo hexo init
  • window下：
    • 进入blog文件夹在，鼠标右键打开git bsh
    • 输入hexo init

9.查看blog文件夹

  • linux下：ls

  • window下：dir

10.启动本地博客服务

hexo s

进入浏览器打开：http://localhost:4000/

11.创建文章

hexo n

12.使用typro就可以编辑.md文件了，使用makedown语法

13.清理

hexo clean

14.生成

hexo g

15.再次启动本地博客服务，刷新网页就可以看到我们刚写的文章了

2.部署到github上

1.进入github，创建一个新的仓库

创建Repository name，其中Primaryo是github账户名，后面.github.io是固定的，这个neme就是以后网站访问的域名

2.在blog目录下安装git部署插件

npm install hexo-deployer-git –save

3.配置_config.yml

可以使用Notepad++打开，文件最后面

4.部署到远程Github仓库

  • 清理 hexo c
  • 生成hexo g
  • 部署hexo d

    git config –global user.email “xxx”
    git config –global user.name “xxx”
